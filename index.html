第1章① ヨーロッパ人の海外進出（大航海時代）

大航海時代の背景

  • 目的：香辛料を求めてアジアへ直接行きたい
  • → イスラーム商人を介さず利益を独占したい
  • 羅針盤・造船技術・地図の発達が後押し
  • 結果：世界の一体化が進む

主な探検家と業績

  • コロンブス（スペイン支援）1492年 アメリカ大陸到達
  • ヴァスコ・ダ・ガマ（ポルトガル）インド航路開拓（1498年）
  • マゼラン 世界周航（1519〜22年）
  • ディアス（喜望峰到達）・カボット も重要

トルデシリャス条約（1494年）

  • スペイン と ポルトガル が締結
  • 大西洋に分界線を設定→世界を二分
  • 東（アジア・アフリカ）→ポルトガル
  • 西（アメリカ）→スペイン（ブラジルは例外でポルトガル）

コロンブス交換・影響

  • 新大陸 → 欧：トマト・じゃがいも・とうもろこし
  • 欧 → 新大陸：天然痘などの伝染病・馬・小麦
  • 先住民が激減（伝染病の影響大）
  • 銀の大量流入 → ヨーロッパで価格革命

三角貿易・プランテーション

  • 欧州→アフリカ（武器・雑貨）→アメリカ（奴隷）→欧州（砂糖・綿花）
  • 1200万人もの奴隷が連行されたとも
  • サトウキビ農場（プランテーション）で強制労働
  • → イギリスが資本を蓄積 → 産業革命の資金源へ

産業革命（18世紀後半〜）

  • イギリスが世界に先駆けて開始
  • 蒸気機関・紡績機などの機械が発明される
  • 綿織物工業 → 鉄鋼・重工業へ発展
  • 「イギリスは世界の工場」と呼ばれる
第1章② 宗教改革・科学革命・啓蒙主義

ルターの宗教改革（ドイツ）

  • 聖書（神の言葉）が最高権威
  • 信仰のみによって救われる
  • 免罪符（贖宥状）を痛烈に批判
  • → プロテスタント（新教）の誕生

カルヴァンの宗教改革（スイス）

  • 予定説：救われる人は神があらかじめ決めている
  • 労働・蓄財を肯定 → 資本主義精神と親和
  • フランス（ユグノー）・オランダ・スコットランドに普及

対抗宗教改革 / イエズス会

  • カトリック側の内部改革運動
  • イエズス会設立（ロヨラ）
  • ザビエル：日本・アジアへ布教
  • 善行・教育も重視（啓発的）

科学革命 → 啓蒙主義

  • 科学革命：経験・実験による自然法則の発見
  • 真理は聖書ではなく人間の経験で得られる
  • → 啓蒙主義：理性で社会を改革しよう
  • → フランス革命・アメリカ独立宣言の思想的基盤へ
第1章③ 主権国家体制・絶対王政・イギリス議会政治

主権国家体制とは

  • 王に権力を集中させる中央集権化
  • 当時は地方分権（諸侯が権力をもつ）が一般的
  • 神聖ローマ帝国は諸侯が分立→主権国家体制が遅れていた
  • → 各国が主権国家体制をめざして競争

絶対王政（フランス）

  • ルイ14世「朕は国家なり」（17〜18世紀）
  • 王に全権力が集中する絶対王政の典型
  • ベルサイユ宮殿建設（貴族を宮廷に集め弱体化）
  • 王権神授説：王の権力は神から与えられた

イギリス議会政治の歩み①

  • ジェームズ1世：王権神授説を主張（議会と対立）
  • チャールズ1世：ピューリタン革命で処刑（1649年）
  • クロムウェル：共和政を樹立・独裁
  • 王政復古 → チャールズ2世・ジェームズ2世

イギリス議会政治の歩み②

  • 名誉革命（1688年）：議会がジェームズ2世を追放
  • ウィリアム3世・メアリ2世を迎える（無血）
  • 権利の章典（1689年）：議会の権限を明確化
  • → 立憲君主制・議院内閣制の確立

権利の章典 主な内容

  • 議会の同意なく法律停止・課税は不可
  • 議員の発言・討論は自由
  • 議会の定期的開催を義務化
  • → 王が議会を無視して政治できない

ロシアの動向

  • 南下政策：暖かい海（不凍港）を求めて南へ
  • 北極に近い海は凍っており、船が通れない
  • ネルチンスク条約（1689年）：ロシアと清（中国）の国境画定
第2章① アメリカ独立革命

背景・七年戦争

  • 北アメリカ大陸はイギリス・フランス・スペインの植民地
  • 七年戦争（1756〜63年）：英仏間の植民地争い
  • 戦費がかかり、イギリスが植民地に課税強化
  • 印紙法・茶法→植民地人の強い反発

独立への流れ

課税強化（印紙法・茶法） →
「代表なくして課税なし」
ボストン茶会事件（1773年） →
大陸会議（植民地全体で対抗） →
独立戦争 → 独立宣言（1776年）

独立宣言の内容

  • すべての人は平等に創られた
  • 生命・自由・幸福の追求は奪えない権利
  • 政府の正当な権力は人民の同意に基づく
  • トマス・ペイン「コモンセンス」が世論を後押し

アメリカの政治体制

  • 初代大統領：ワシントン
  • 三権分立：行政（大統領）・立法（議会）・司法（裁判所）
  • 連邦制：各州の自治を認めた上で連邦国家を形成
  • 相互の抑制・均衡（チェック・アンド・バランス）
第2章② フランス革命

旧制度（アンシャン・レジーム）

  • 第一身分：聖職者（約0.5%）特権・免税
  • 第二身分：貴族（約1.5%）特権・免税
  • 第三身分：平民（約98%）重税を一手に負担
  • 財政悪化 → 貴族にも課税しようとして反発

革命の流れ

財政危機 → 全国三部会招集（174年ぶり）
→ テニスコートの誓い
→ バスティーユ牢獄襲撃（1789年7月14日）
→ 人権宣言採択
→ 国王処刑 → 第一共和政

人権宣言の内容

  • 人間は自由かつ平等に生まれる
  • 主権は国民にある
  • 自由・所有・安全・圧政への抵抗は権利
  • 所有権は神聖かつ不可侵

フランス革命の意義

  • 旧体制（封建制・身分制）の打破
  • 自由・平等・国民主権の思想を制度化
  • 革命思想がヨーロッパ各国に波及
  • アメリカ独立革命と相互に影響しあった
第2章③ ナポレオン

ナポレオンの台頭

  • 本名：ナポレオン・ボナパルト
  • 軍人として頭角を現しクーデターで権力掌握
  • 第一帝政（皇帝ナポレオン）樹立
  • ナポレオン法典：法の下の平等・私有財産の保護・契約の自由

ナポレオンのヨーロッパ支配

  • 軍事力でヨーロッパ各国を征服・支配
  • 大陸封鎖令：イギリスとの貿易を禁止（1806年）
  • フランス革命の理念（自由・平等）をヨーロッパへ広める
  • → 各国でナショナリズムが目覚める

失敗・没落の流れ

大陸封鎖令でヨーロッパ各国も困窮
→ ロシア遠征（1812年）失敗
　（冬の寒さ＋焦土作戦で壊滅）
→ 国力低下・反ナポレオン同盟
→ ワーテルローの戦い（1815年）敗北
→ セントヘレナ島に島流し

ナポレオン敗北の理由

  • イギリスは島国 → 軍事征服できない
  • 大陸封鎖でヨーロッパ各国の経済も打撃
  • ロシアの広大な国土と厳冬・焦土作戦
  • 征服された各国でナショナリズムが高まり抵抗
